Tricks to test Titanium Apps

  • Avoid global events and using the titanium namespace for storing things unless necessary
  • Use commonjs modules for everything
  • In the event callbacks, use functions that you can call anytime you want, that way it is a lot easier to share common behaviours and call them anywhere you want
  • Reuse code
